# Cybersecurity Chatbot Console Application

This is part 1 of the POE where we had to create a console application for a cybersecurity chatbot.

## The features of the chatbot are:
1. user interaction: the chatot will will ask for the users name before contiuing, which will go hand in hand with the voice greeting feature.
2. voice greeting: integrated welcome audio for the user.
3. user interface: the chatbot consists of an typewriter effect and ASCII art for an engaging experience.
4. responses: the chatbot will provide responses based on the user 's input, with a focus on cybersecurity topics or questions.

## How to run the application:
There is a play button at the top, the color is green, click on it to run the application. You can also use the shortcut key (CTRL+F5) to run the application.

## Project Structure:
Responses.cs: Stores simple conversation responses and returns the right reply based on what the user has typed.
UIHelper.cs:Handles the chatbot’s display features like showing text, playing sounds, and creating a typing effect for responses, along with an ascii image.
CyberQuestions.cs: handles/stores cybersecurity questions and answers and finds the right answer based on what the user types.
Program.cs: Main entry point and chat loop logic, basically the main file of the application.

## Built with:
This is a console application built using C# and .NET 10.0.
I also used Visual Studio as the IDE for building and running the application.
